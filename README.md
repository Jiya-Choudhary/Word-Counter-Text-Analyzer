# 📊 Word Counter & Text Analyzer

A Python command-line application that analyzes user-provided text and extracts useful insights.

## ✨ Features

* 📝 Counts total words
* 🔍 Finds the longest word
* 🔤 Identifies unique vowels present in the text
* ⚡ Fast and lightweight implementation

## 🛠️ Tech Stack

* Python 3

## 🚀 How to Run

```bash
python word_counter.py
```

## 📌 Example

```text
Enter Paragraph: Python is an amazing programming language

Number of words: 6
Longest word: programming
Vowels found: ['o', 'i', 'a', 'e']
```
